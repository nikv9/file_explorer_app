### File Explorer

To run this app follow the given below steps -

- After clone the app from github run these commands -

```
npm install
npm run dev
```

### Feature of the App

- You can add new folder
- You can add new file and content
- You can see the selected file content in right panel
- Error implementation if required fields are missing while adding new file/folder

### Used feature of React.js

- Redux,
- uuid,
- tailwind css,
- redux toolkit,
- react-toastify
